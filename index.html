О разработке

  • Python Django

    Сайт реализован на популярном Python фреймворке Django. Благодаря таким технологиям сайт работает быстро и удобно как для пользователей так и для администратора

  • Bootstrap

    Визуальные решения нашего сайта оформлена с помощью веб-фреймоврка Bootstrap. Именно поэтому мой сайт настолько красив и выполнен в уникальном едином стиле

  • Postman & RestFul Api

    Также сайт поддерживает RestFul Api, что позволяет работать с GET PUT POST запросами (и все это протестировано с помощью Postman)

  • Импорт/Экспорт

    Также сервис предлагает уникальную возможность для администратора по импорту и экспорту данных с помощью таблиц! Это дейвствительно удобно
